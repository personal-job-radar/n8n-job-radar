Personal Job Radar

Personal Job Radar is a private n8n automation project created to help an individual user find job opportunities relevant to their professional skills and preferences.

How it works

The workflow retrieves job search results from authorized data sources, removes duplicate listings, evaluates their relevance and sends private notifications to the user's Telegram account.

Planned Jooble API usage

The Jooble REST API will be used only for a private and personal job search. Requests will be limited in frequency and based on selected job titles, skills and locations.

Data and privacy

Job listings will not be sold or republished as a public job board. Candidate CV documents, telephone numbers and other sensitive personal information will not be published on this website.

Technology

The project uses n8n for workflow automation and Telegram for private notifications.
